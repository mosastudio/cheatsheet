* class instances are passed by reference; structs are passed by value. 

* Int

* Double

* String

* Array

```swift
  var myArr1 = [String]()
  var myArr2: [String] = []
```

* Dictionary

```swift
  var dict1 = [String: String]()
  var dict1: [String: String] = [:]
```

* Set

```swift
  var set1 = Set<String>()
  var set2: Set<String> = []
```
